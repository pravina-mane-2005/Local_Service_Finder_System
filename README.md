# 🏠 Local Service Finder System

## 📌 Project Overview
The Local Service Finder System is a web-based application developed using **Spring MVC**. It helps users find local services such as plumbers, electricians, and other service providers. The system has **three panels**: **User**, **Service Provider**, and **Admin**, making management and usage simple and efficient.

---

## 🛠️ Technologies Used
- Java  
- Spring MVC  
- JSP, Servlets  
- MySQL  
- HTML, CSS  
- Apache Tomcat  

---

## ✨ Features

### User Panel
- 🔍 Search for local services by category or location  
- 📄 View service provider details and contact information  
- 📝 Request services 

### Service Provider Panel
- 🏷️ Register as a service provider  
- 🛠️ Manage service listings  
- 📬 Respond to user requests  

### Admin Panel
- 👤 Manage users and service providers  
- ✅ Approve or reject service listings  
- 📊 Generate reports or analytics 

---

## 🗄️ Database Details
**Database:** MySQL  

**Tables:**  
- Users  
- ServiceProviders  
- Services  
- Requests
